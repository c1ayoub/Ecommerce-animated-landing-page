# 🍯 Absolute Amlou — Premium Moroccan Spread Landing Page


## ✨ Features

- **Smooth Scrolling**: Integrated with **Lenis** and **Locomotive Scroll** for a fluid, luxurious feel.
- **Dynamic Animations**: Powered by **GSAP** (GreenSock), featuring text reveals, parallax effects, and scroll-triggered canvas sequences.
- **Immersive Visuals**: High-resolution imagery and sophisticated typography using **Cormorant Garamond** and **Inter**.
- **Interactive Stacks**: Benefit cards and process sections that respond dynamically to user interaction.
- **Fully Responsive**: Crafted to look stunning from 4K displays down to mobile devices.

---

## 🛠️ Tech Stack

- **Framework**: [Vite](https://vitejs.dev/)
- **Animation**: [GSAP](https://greensock.com/gsap/)
- **Typography**: [Split-type](https://github.com/luke-peacock/split-type)
- **Scrolling**: [Lenis](https://github.com/darkroomengineering/lenis) & [Locomotive Scroll](https://locomotivemtl.github.io/locomotive-scroll/)
- **Styling**: Vanilla CSS (Modern CSS properties & Variables)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/) or [Bun](https://bun.sh/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/absolute-amlou.git
   cd absolute-amlou
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   bun dev
   ```

4. **Build for production**
   ```bash
   npm run build
   # or
   bun build
   ```
